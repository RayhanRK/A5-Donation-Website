Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents a webpage as a tree structure, allowing developers to modify its content and structure using JavaScript.

How do you select an element from the DOM?

We can select a DOM element using methods like: document.getElementById(), document.querySelector(), or document.getElementsByClassName()

What is event Delegation in the context of the DOM, and why it is useful?

Event delegation is a technique in the DOM where a single parent element handles events from multiple child elements. It's useful because it reduces memory usage and simplifies code by allowing us to manage events efficiently for dynamically added elements.

How do you manipulate an elements attributes and styles using the DOM?

By applying the following methods we can manipulate elements' attributes and styles using the DOM are :
1. getAttribute()
2. setAttribute() and
3. removeAttribute()
we can also use element.style to modify CSS properties (example: element.style.color = 'blue').
